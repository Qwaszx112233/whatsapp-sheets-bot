# TERMINAL COMMANDS — WAPB

Этот файл — шпаргалка для работы с проектом из терминала VS Code.

---

## 1. Что является главным источником правды

Главный источник правды по проекту — **локальная папка в VS Code**.

Основные исходники для Google Apps Script:
- `*.gs`
- `*.html`
- `appsscript.json`

Если рядом лежат `*.js` и `*.gs`, по умолчанию **канон = `*.gs`**, пока отдельно не описан build-процесс.

---

## 2. Где что делать

### VS Code
Здесь редактируются:
- `.gs`
- `.html`
- `.json`
- `.ps1`
- `.md`

### Терминал VS Code (PowerShell)
Здесь запускаются:
- `git`
- `node`
- `npmps`
- `clasp`
- `gas-pull`
- `gas-push`
- `gas-watch`
- `git-save`
- `git-sync`

### Google Apps Script
Здесь:
- запускаются GAS-функции вручную
- смотрятся execution logs
- проверяется runtime
- делаются только аварийные правки, если очень надо

### GitHub
Это удалённый репозиторий локального проекта из VS Code.
